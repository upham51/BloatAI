## File Requirements

- **Format**: PNG (preferred for web)
- **Optimization**: Images should be optimized for web (compressed but maintaining quality)
- **Responsive**: Will automatically scale to fit mobile screens
- **Accessibility**: Alt text is provided in the component

## How to Add Images

1. Rename your uploaded images to match the filenames above
2. Place them in this directory (`/public/assets/images/`)
3. The BloatingGuide component will automatically display them

## Image Locations in Guide

- **Infographic #1**: Appears in "What Exactly is Bloating?" section, after the intro paragraph
- **Infographic #2**: Appears in "The Food Culprits (FODMAPs)" section, under "What Happens Step-by-Step"
- **Infographic #3**: Appears in "The Food Culprits (FODMAPs)" section, under "Common High-FODMAP Foods"

## Troubleshooting

If images don't appear:
1. Check that filenames match exactly (case-sensitive)
2. Verify images are in `/public/assets/images/` directory
3. Clear browser cache and refresh
4. Check browser console for any loading errors
