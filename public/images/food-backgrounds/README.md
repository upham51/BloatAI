# LOW FODMAP Food Background Images

This directory contains beautiful, luxurious food images for the dashboard card backgrounds.

## Image Specifications

### Technical Requirements:
- **Format**: WebP (for fast loading) or high-quality JPG
- **Dimensions**: 800x600px or 1200x800px (landscape orientation)
- **File Size**: Keep under 200KB each (use compression)
- **Naming**: food-bg-1.webp, food-bg-2.webp, ... food-bg-8.webp

### Design Requirements:
- **Style**: Luxurious, appetizing, professionally styled food photography
- **Lighting**: Warm, natural lighting with soft shadows
- **Composition**: Clean, minimalist plating on neutral backgrounds
- **Color Palette**: Earth tones, warm browns, gentle greens, soft whites
- **Text Contrast**: Slightly darker or muted backgrounds work best for white text readability
